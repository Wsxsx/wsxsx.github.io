Android_Studio安装

发表于2023-06-04|更新于2023-06-06
|字数总计:524|阅读时长:1分钟|阅读量:

学校的移动应用开发课程，不得已下一个，记录一下
还是得吐槽一下老师，绝了，混子

下载

  1. 官网下载

    官网：https://developer.android.google.cn/studio

  2. 随便在哪下，跑起来就行不是吗

我下的版本是2022.1

安装

  1. 打开安装包点击next

  2. 继续next，打勾那个是安卓虚拟设备，记得勾上

  3. 更换安装位置，别做c盘战士就是了，继续next

  4. Install，左下角打勾的地方是不创建快捷方式，一般没必要

  5. ok

  • 打开Android Studio

  • 安装谷歌什么的，不需要

下载SDK

SDK全称为Software Development kit,就是软件开发工具包,可以将app源码编译为可执行的app应用

  • 提示需要安装SDK

  • 选 custom

  • 选界面，进去也可以调，随意选

  • 安装SDK，除了安装位置，其他默认即可 （看着我的眼睛，不要做c盘战士好吗）

  • 这里是虚拟机的内存，学习两个G也够用了

  • 提示要安装这个，next即可

  • 要下载SDK，next即可

  • 一定一定要选Accept，最后一个好像是Intel硬件加速什么的，我也点了，前两个必选Accept,选完Finish

  • 稍等一会，安装中

  • 下载完点击finish，进入到新建工程界面

这就差不多ok了

新建项目

  • 这里新建一个项目

  • 选这个

  • 默认位置改一下，默认语言改为java，然后finish

  • 然后就会默认创建一个工程

安装安卓模拟器，

  • 点击Device Manager

  • 点击，都是外观，我这里选择的是 Pixel 4，选择完 next

  • 版本我选择的13

  • 安装，点击下载键

  • 无脑finish

  • 点击启动键

  • 如果想要这种界面的模拟器，不需要就略过

    左上角点击File -> 点击 Settings或者Ctrl + Alt + s -> 搜索栏搜索 Emulator -> 将 Launuch in a tool window 的复选框对勾去掉 -> 设置框右下角 apply -> ok -> 重新启动 Android Studio 即可

  • 重启完成后运行模拟器，然后点击

  • 效果如图,ok了
