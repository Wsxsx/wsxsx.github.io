虚拟机安装运行环境

发表于2023-06-02|更新于2023-06-06|编程学习
|字数总计:1.6k|阅读时长:6分钟|阅读量:

前言

  • 安装过程中可能会出现一些问题，不对，是肯定会出问题，但是不要放弃，出问题就去解决它，我也重装了很多次，大不了重头再来！反正电脑又不会爆炸不是吗💕

使用 Vmware8 + Centos7 环境安装 JDK17、tomcat8、mysql8、lrzsz等环境

安装JDK17

  • 一般软件我都默认安装在 /usr/local 下

下载JDK两种方式

1. 通过 wget https://download.oracle.com/java/17/latest/jdk-17_linux-x64_bin.tar.gz 直接在shell下载
2. 通过官网下载，然后再通过Xshell或者finalShell之类的工具上传到linux  
    网址：https://www.oracle.com/cn/java/technologies/downloads/#java17

安装

  • 我用的是第二种方式
  • 首先进入到 usr/local
  • 创建一个存放 JDK17 的目录

  • 押解JDK17包

  • 押解完毕后,vim/etc/profile

  • 配置环境变量

  • 红框里的路径是你的jdk目录下的路径，所以如果你在其他的文件下压解jdk17那么就需要你自己去pwd一下查看路径

  • 修改后ESC保存

  • 重启环境变量

查看版本号

PS：如果有其他的问题就基本上就是环境变量没有配置好或者误删除掉了里面的符号，建议重新删掉然后再去配置。

  • 至此JDK17就安装完成咯！

tomcat安装

tomcat版本：8.0.50

点击【Which version】可查看Tomcat各版本所对应的jdk版本，（下载Tomcat各版本之前需要查看自己所对应的jdk版本，如果你下载的Tomcat版本高于或低于所下载的jdk版本，安装运行之后可能会出现报错，就比如我下载的是如图以下的Tomcat8.5的版本，那么jdk的版本至少得是Java7的版本或者Java7之后的版本都行，如果我下的是Tomcat11的版本，那么jdk版本就至少是Java17的版本或Java17之后的版本。）

后缀 含义
Core 是Tomcat正式的二进制发布版本
zip 压缩包。
tar.gz Linux操作系统和Unix操作系统形式的压缩包
32-bit Windows zip Windows32位操作系统压缩包
64-bit Windows zip Windows64位操作系统压包
32-bit/64-bit Windows Service Installer Windows32位与64位操作系统的安装程序包
  • 下载tomcat安装包
  • http://tomcat.apache.org/
  • 我这边是下载的apache-tomcat-8.0.50.tar.gz


  • 解压

  • 我是放在了usr/local/tomcat下所以要创建tomcat文件夹并将解压文件移动到tomcat下(由于之前新建过再次创建或出现重复)

  • 一样，先创建一个目录

  • 进入tomcat安装bin目录
  • 启动tomcat

  • 成功启动后浏览器输入http://localhost:8080/查看信息（输入自己的IP，比如我输入服务器IP47.97.189.117:8080）

  • 那么tomcat也就安装完成了

安装mysql

  • 最麻烦的来了

下载mysql

Mysql安装版本：8.0.33

  • 进入官网下载对应的版本
    网址 https://dev.mysql.com/downloads/mysql/

安装mysql之前先做的事(必看!必看!必看！)

  1. 查看本地有没有安装mariadb ，有的话删除

  2. 查看本地有没有安装Mysql

  3. 查找本地有没有Mysql文件

  4. 确保环境准备无误

  5. 环境准备没问题开始上传Mysql安装包安装（rz/xftp）

  6. 创建一个mysql目录存放解压出来的安装包

  7. 解压缩

  8. 查看解压后文件夹内容

  9. 此时你的文件夹应该是跟我一样的

  10. 开始安装依赖按照顺序安装否则报错!!!!!!!!!

  11. 安装完成后, 开始进行下一步

  12. 输入mysql密码(这里输入密码是不显示的，但是是输入进去了)

  13. 进入数据库以后要修改密码(毕竟他给的密码太麻烦了不是吗)

  • 至此完成

  • 设置远程连接

然后用我们的远程工具测试即可

对了，navicat或者其他什么测试记得linux连接上数据库

安装 lrzsz

  1. 搜索lrzsz安装包

  2. 使用yum命令在线安装

Yum(全称为Yellow dog Updater, Modified)是一个在Fedora和RedHat以及Cent05中的Shell前端软件包管理器。基于RPM包管理，能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖关系，并且一次安装所有依赖的软件包，无须繁琐地一次次下载、安装。

  • 安装完后输入 rz ，就会跳出一个窗口，选择文件后即可上传
