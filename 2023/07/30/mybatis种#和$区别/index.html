mybatis种#和$区别

发表于2023-07-30|更新于2023-07-30|编程学习
|字数总计:334|阅读时长:1分钟|阅读量:

#{} 自带 ‘ ‘ 在sql语句中作为 ?
${} 不带 ‘ ‘ 在sql语句中作为 本身 要注意字符串拼接

批量删除只能只有用 ${} ,使用 in关键字
动态表名只能使用 ${}, 否则会报错
select * from user where name = '${param1}' and password = '${param2}'

自定义哈希map键值对
接口中新增
User loginByMap(HashMap<String,Object> map);
2. xml映射文件添加

3. 测试
@Test
public void loginByMap(){
SqlSession sqlSession = SqlSessionUtil.getSqlSession();
UserMapper mapper = sqlSession.getMapper(UserMapper.class);
HashMap<String, Object> map = new HashMap<>();
map.put(“name”,”TOM666”);
map.put(“password”,”123”);
User user = mapper.loginByMap(map);
System.out.println(user);
sqlSession.close();
}
测试，成功

  • 如果mapper接口方法的参数是实体类类型的参数
    只要通过 #{} 和 ${} 访问实体类的属性名，就可以获取相对应的属性值 insert into user values (null,#{name},#{password},#{age},#{tel})

@Test
public void save(){
SqlSession sqlSession = SqlSessionUtil.getSqlSession();
UserMapper mapper = sqlSession.getMapper(UserMapper.class);
User user = new User(null, “adi”, “123”, 18, “sdas”);
int i = mapper.save(user);
System.out.println(i);
sqlSession.close();
}
