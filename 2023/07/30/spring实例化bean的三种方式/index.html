spring实例化bean的三种方式

发表于2023-07-30|更新于2023-07-30
|字数总计:457|阅读时长:2分钟|阅读量:

实例化bean的三种方式
使用无参可访问的构造方法(可写可不写)

public BookDaoImpl() {
System.out.println(“book Dao constructor”);
}
使用静态工厂

工厂类

package com.yin.factory;
​
import com.yin.dao.OrderDao;
import com.yin.dao.impl.OrderDaoImpl;
​
public class OrderFactory {
public static OrderDao getOrderDao() {
for (int i = 0; i < 10; i++) {
System.out.print(i);
}
return new OrderDaoImpl();
}
}
使用 factory-method 指定静态工厂调用的方法

启动类

OrderDao orderDao = OrderFactory.getOrderDao();
orderDao.save();
使用实例工厂

dao接口

package com.yin.dao;
​
public interface UserDao {
void save();
}
dao实现层

package com.yin.dao.impl;
​
import com.yin.dao.UserDao;
​
public class UserDaoImpl implements UserDao {
@Override
public void save() {
System.out.println(“user Dao”);
}
}
factory工厂,这里不是静态的

package com.yin.factory;
​
import com.yin.dao.UserDao;
import com.yin.dao.impl.UserDaoImpl;
​
public class UserFactory {
public UserDao getUserDao(){
return new UserDaoImpl();
}
}
xml配置bean

​


​


测试

package com.yin;
​
import com.yin.dao.UserDao;
import org.springframework.context.ApplicationContext;
import org.springframework.context.support.ClassPathXmlApplicationContext;
​
public class App3 {
public static void main(String[] args) {
ApplicationContext c = new ClassPathXmlApplicationContext(“applicationContext.xml”);
UserDao userDao = (UserDao) c.getBean(“userDao”);
userDao.save();
}
}
方式三变种，使用FactoryBean实例化

创建UserDaoFactoryBean类,继承FactoryBean接口，实现泛型

package com.yin.factory;
​
import com.yin.dao.UserDao;
import com.yin.dao.impl.UserDaoImpl;
import org.springframework.beans.factory.FactoryBean;
​
public class UserDaoFactoryBean implements FactoryBean {
//代替原始实例工厂中的创建对象的方法
@Override
public UserDao getObject() throws Exception {
return new UserDaoImpl();
}
​
// 返回对象类型
@Override
public Class<?> getObjectType() {
return UserDao.class;
}
}
配置xml文件，最后实例化出来的不是
FactoryBean对象，而是FactoryBean中的getObject方法造出来的对象


其余照旧

默认为单例模式，如果要修改为多例,在
factoryBean类中是实现isSingleton方法
/**
*
* @return true：单例 ，false:多例
*
*/
@Override
public boolean isSingleton() {
return false;
}
